* Web Security / AppSec
* OWASP Top 10
* Burp Suite: Proxy, HTTP History, Repeater, Decoder
* SQL Injection, XSS, IDOR, Broken Access Control
* Authentication / Session Management
* Nmap enumeration
* Wireshark traffic analysis

* Security reporting: finding, severity, evidence, impact, remediation

## Проекты

### OWASP Juice Shop Security Assessment

Учебный security assessment локального стенда OWASP Juice Shop.
Проверялись authentication, access control, input handling, file exposure, API behavior, error handling и CORS.

Основные темы:

* SQL Injection
* Admin panel access
* DOM-based XSS
* Directory listing
* Weak password recovery
* JWT analysis
* User enumeration
* CORS misconfiguration

### OWASP Mutillidae II Security Assessment

Учебный security assessment локального Docker-стенда OWASP Mutillidae II.

Основные темы:

* SQL Injection
* Reflected XSS
* Stored XSS
* Command Injection
* Local File Inclusion
* CSRF
* PHP Info disclosure
* Security headers / session cookies

### Website Security Assessment — Sanitized

Обезличенная версия отчёта по ограниченной black-box проверке безопасности сайта.

Проверка выполнялась в безопасных рамках:

* без DoS/DDoS;
* без подбора паролей;
* без изменения данных;
* без доступа к персональным данным;
* без эксплуатации за пределами согласованного scope.

### Crypto Company AWS Security Baseline — Sanitized

Обезличенный мини-отчёт по базовым мерам защиты AWS-инфраструктуры для крипто-компании.

Темы:

* IAM / MFA / least privilege
* Root account protection
* CloudTrail / CloudWatch
* RDS backups / Multi-AZ / Point-In-Time Recovery
* Encryption
* Private subnets / network segmentation
* Incident Management
* Vendor Risk Management
* Business Continuity Plan
